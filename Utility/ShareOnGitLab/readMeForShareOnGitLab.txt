%% Overview
% The gitlab.jar (by Mark Robinson) file contained in this folder is for sharing Simulink   % Projects on the MathWorks GitLab server, via the Share to GitHub extension for R2015a. The % extension makes it really simple to create a repository on the MathWorks GitLab server and % push your project to it. All from the Simulink Project Share menu. It works just like the % 15a Share to GitHub feature, except your project ends up on our internal GitLab server. A % feature video for sharing to GitHub can be found at:
%
% http://inside-files/public/Mark_Robinson/github/github.mp4
%
%
% The full path to gitlab.jar just needs to be placed in a file named javaclasspath.txt     % within the 15a prefdir folder (C:\Users\dmeissne\AppData\Roaming\MathWorks\MATLAB\R2015a)
% 
% 
% If gitlab.jar is somehow lost or deleted, download it from:
